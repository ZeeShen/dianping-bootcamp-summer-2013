## FAQ

### 如何在本地浏览演示资料

首先，需要能够在项目根目录起一个本地 server。

如果安装并配置好了 node.js，可进行如下操作

```bash
npm install anywhere -g
cd where/you/put/to/
anywhere
```

然后，即可访问 [http://localhost:8080/presentation/](http://localhost:8080/presentation/)
